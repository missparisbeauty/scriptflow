使用流程說明

  1. 步驟 1：在「② 腳本/分鏡」生成腳本後，複製 腳本 ID
  2. 步驟 2：發布到脆/IG，把貼文網址貼到下方「新增發布記錄」儲存（會得到一個 追蹤 ID）
  3. 步驟 3：發布後第 7 天 / 第 14 天，去脆/IG 後台抄下成效數據（觀看數、按讚、留言、CTR…）
  4. 步驟 4：用「更新成效數據」表單把數字填回來
  5. 步驟 5：累積 ≥5 支有 7 天成效的作品 → 在「④ 品牌 DNA」按計算，會推導出爆款公式

⚠️ ScriptFlow 不會自動爬脆/IG 的成效（沒 Meta API 權限），所以數據要手動輸入。日後你可以串你自己的 Meta Business API 自動化。
