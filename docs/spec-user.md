# spec-user.md — ScriptFlow 短影音變現操盤台

---

## § 產品定位

**ScriptFlow** 是一套品牌內部小編專用的短影音變現操盤台。

每天早上 09:00 自動從小紅書、抖音、Threads 抓取熱門爆款，分析流量節奏，產出適合發布到 Threads 和 Instagram 的腳本與分鏡，並追蹤發布後的實際成效，讓 AI 學習品牌觀眾偏好，持續優化下次的腳本品質。

**解決的問題：**
- 小編每天耗時手動找爆款、拆解流量邏輯、再從頭寫腳本
- 腳本寫完沒有系統追蹤成效，不知道哪種開頭、CTA 對自家觀眾最有效
- 美妝類爆款節奏好學，但硬套到髮品會導致主題偏離、帶錯貨

**目標使用者：** 品牌內部小編（單一品牌，單人或小組使用）

---

## § 核心功能

### 1. 每日爆款自動爬取與候選推薦

系統每天台灣時間 09:00 自動啟動，從小紅書、抖音、Threads 抓取過去 24 小時的熱門內容，篩選出今日推薦的 3 個爆款候選。

小編登入後看到：
- AI 自動判定的**今日主題**（例：受損髮質修護）及主題集中度
- 3 個爆款候選卡，每張標示平台來源、互動數、完播率、主題契合度、導購意圖密度、漏斗位置（種草型 / 拉流型 / 收割型）

**篩選邏輯（兩種策略可切換）：**
- 三平台各抽 1（互補視角，推薦）
- 混合熱度 Top 3（純互動量排序）

判定標準：互動量（讚 + 留言 + 分享）× 完播率（抖音／IG 加權）× 導購意圖密度（留言中購買關鍵字比例）

### 2. 雙軌爬取（髮品模式）

切換至「髮品」分類時，系統同時執行兩條爬取軌道：

- **A 軌**：直接爬取髮品爆款（洗髮精、護髮素、髮膜、造型品、頭皮保養），學習帶貨語言
- **B 軌**：爬取美妝爆款（固定抓護膚保養教學 + 產品開箱實測兩類，AI 每日評估其他類型的結構相似度，≥ 80% 才納入，彩妝教學等低相似度類型自動排除），僅學習流量節奏

小編可在 B 軌篩選區手動開關 AI 推薦的類型，或輸入關鍵字新增（AI 仍會評分，低於 80% 提示警告）。

### 3. 手動補充爆款連結

小編可在候選區下方手動貼入任何平台的爆款連結，加入分析。

### 4. 腳本產出（三個版本）

點擊「生成腳本」後，AI 從 3 個爆款萃取共通元素，產出同一主題的三個版本：

| 版本 | 格式 | 漏斗角色 | CTA 策略 |
|---|---|---|---|
| Threads 貼文 | 純文字短文（約 100 字以內） | 種草型 | 引留言 → 導 Line 社群 |
| Threads 短影音（脆） | 30 秒口播腳本（4 段時間軸） | 拉流型 | 「你們也這樣嗎？」→ 導 Line 社群 |
| Instagram Reels | 60 秒口播腳本（4–5 段時間軸）+ Caption / Hashtag | 收割型 | 「限動連結搶最後 3 組」直接成交 |

每個腳本卡包含：
- 漏斗角色標記（種草 / 拉流 / 收割）
- **3 種 CTA 變體**（供小編 A/B 測試，標明預期效果：引留言 / 直接導購 / 引追蹤 / 名單收集）
- **合規檢查**：自動掃描禁詞（抖音、IG、Threads 各平台敏感詞），違規詞標橘色警告並提示改寫建議

腳本可直接在網頁複製，或下載 Word 檔。

### 5. 分鏡腳本（兩個平台版本，可切換）

分鏡腳本以四欄表格呈現，依平台節奏分為兩個版本，上方點擊切換：

| 欄位 | 說明 |
|---|---|
| 秒數 | 鏡頭編號與起訖時間 |
| 畫面呈現 (Visual) | AI 示意圖（預留位）+ 取景與構圖文字描述 |
| 聲音與音效 (Audio) | BGM 說明 + 音效（SFX）逐一標注 |
| 腳本口白 (Subtitle) | 講者口白或字卡說明 |
| 產品露出策略 | 每個鏡頭的產品曝光方式（不露 / 場景帶過 / 主動特寫 / CTA 不露產品） |

脆版 4 鏡（30 秒），IG Reels 版 5 鏡（60 秒），各自對應不同節奏邏輯。

分鏡稿可下載 PDF 或 Word。

### 6. 風格分析

顯示三個爆款所代表的三個平台的流量特徵拆解：
- 小紅書：開頭句型、標籤密度、導流方式
- 抖音：鉤子結構、卡點節奏、完播策略
- Threads：口語風格、段落結構、互動引導

### 7. 成效追蹤與品牌爆款 DNA

小編發布影片後，回到成效追蹤頁貼入發布連結，系統 7 天 / 14 天後回收觀看數、完播率、CTR、轉換數。

系統累積品牌歷史成效，沉澱為**品牌爆款 DNA**：
- 最佳開頭句型
- 最佳 CTA 組合（脆「Line 社群」+ IG「限動 3 組」勝出率最高）
- 最佳產品露出時機（50% 進度 + 對比畫面）

下次生成腳本時，AI 同時參考「外部爆款 + 自家歷史成效」雙重輸入。

## § 業務規則

### 自動排程
- 每天台灣時間 09:00 自動啟動爬取，不需小編手動觸發
- 排程狀態顯示於頁面右上角，顯示下次執行倒數

### 分類與品牌邊界
- 目前支援三個分類：美妝、美食、髮品
- 髮品模式下，B 軌美妝爆款僅供學習流量節奏，不影響腳本的產品語言與主題
- B 軌彩妝教學、眼妝等結構相似度 < 80% 的類型自動排除，不可人工強制啟用超低相似度類型

### 腳本產出邏輯
- 每次以 3 個爆款為輸入，產出 1 個主題的腳本（不是 3 份）
- 同一份腳本同時產出三個平台版本（Threads 貼文 / 脆 / IG Reels）
- 分鏡稿以口播腳本為基礎，脆與 IG Reels 各自一份

### 漏斗分工（固定策略）
- 脆（Threads 短影音）定位為拉流型：引留言 → 導 Line 社群
- IG Reels 定位為收割型：Line 老客 + 限動連結直接成交
- Threads 純文字貼文定位為種草型：建立信任、引留言互動

### 成效閉環
- 小編需手動貼回發布連結，系統才能回收成效數據
- 品牌爆款 DNA 在累積足夠作品後（建議 5 支以上）開始有意義

## § 外部服務

| 服務 | 用途 |
|---|---|
| 小紅書（非官方爬取輔助）| 抓取美妝 / 髮品熱門貼文與短影音 |
| 抖音（非官方爬取輔助）| 抓取美妝 / 髮品熱門短影音，含完播率 |
| Threads | 抓取台灣市場熱門貼文 |
| OpenAI GPT-5-mini | AI 分析爆款風格、產出腳本、合規檢查、品牌 DNA 沉澱 |
| OpenAI gpt-image-1 | 為每個分鏡產出構圖示意圖（與文字共用同一把 API key） |
| Line 社群 | 脆腳本結尾導流目的地（外部平台，非系統內） |

## § 使用限制與費用

### 目前範圍
- 單一品牌內部使用，無多帳號、無多品牌管理
- 每日自動爬取 3 個候選爆款（可手動補充）
- 腳本語言：繁體中文（台灣市場）
- 圖像生成為示意圖，非正式拍攝參考圖

### 目前不在範圍內（未來可擴充）
- 直播腳本
- 自動發布到各平台
- 多品牌 / 多小編帳號管理
- 成效數據自動回收（目前需小編手動貼回連結）

### 費用
- 預估月費 $5–15 USD（中度使用），主要費用來源為 OpenAI API（GPT-5-mini 文字 + gpt-image-1 分鏡示意圖）
- 雲端運算費用在一般使用量下落在免費額度內
